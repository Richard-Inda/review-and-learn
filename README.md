# Programming Syntax Review

This repository contains notes and practice exercises I am currently working through.

The primary goal is to reinforce language syntax and core programming patterns by writing code manually rather than relying on AI autocomplete or code generation tools.

## Purpose

Modern development environments include extremely powerful AI-assisted tools that can generate large portions of code automatically. While these tools improve productivity, over-reliance on them can weaken direct familiarity with syntax and language fundamentals.

This repository exists as a deliberate practice space to maintain strong recall of:

- Core programming syntax
- Standard library usage
- Common coding patterns
- Language-specific best practices

## Approach

When working through the material in this repository, I follow a few simple rules:

1. Write code manually without AI autocomplete.
2. Attempt solutions from memory before referencing documentation.
3. Use official documentation only after attempting the solution.
4. Use AI tools only for review or explanation after implementation.

The goal is to reinforce understanding rather than maximize speed.

## Repository Structure

The repository will primarily contain:

- Course notes
- Syntax references
- Practice exercises
- Small example programs

Each section will correspond to a specific course, language topic, or concept.
