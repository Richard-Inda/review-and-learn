CSS Notes

Adding CSS to your HTML

3 methods for adding CSS to your HTML:

  1. Inline - adding it directly to the html tag (like any other attribute)
    2. Example: <h1 style="color: red"> This is a red heading
    • Never do this
    • Very limited scope and manual overrides are difficult
  2. Internal - using a style element in the head of the HTML document. Not great
    3. Example: <head> <style>.example{padding-left:20px;}</style></head> This will indent my examples
    • Not reusable
    • Not scalable
    • Not maintainable
  3. External - using a separate CSS file
    4. Example: <head> <link rel="stylesheet" href="style.css"></head> This will link to my style.css file and apply the styles to the page. The lists are styled using them
    • Best practice
    • Modular and reusable
    • Scalable and maintainable

CSS Declarations

Selectors

  • Element Selector
    • Example: <h1> This will select all h1 elements
  • Class Selector
    • Example: .example The . symbol before the class name indicates that this is a class selector.
  • ID Selector
    • Example: #example The # symbol before the id name indicates that this is an id selector
  • Attribute Selector
    • Example: input[type="text"] The [] symbol before the attribute name indicates that this is an attribute selector
  • Pseudo-class Selector
    • Example: input:focus The : symbol before the pseudo-class name indicates that this is a pseudo-class selector
  • Pseudo-element Selector
    • Example: ::before The :: symbol before the pseudo-element name indicates that this is a pseudo-element selector

CSS Properties

Color Properties

  • background-color: Sets the background color of the element
  • color: Sets the color of the text

Font Properties

  • font-family:Sets the font of the text. You can give multiple fonts separated by commas to specify a fallback font.
  • font-size: Sets the size of the text. You can use relative units like em or rem.
  • font-weight: Sets the weight of the text. Like bold, normal, or lighter.
  • font-style: Sets the style of the text. Like italic, normal, or oblique.
  • text-align: Sets the alignment of the text like left, right, center, or justify.
  • text-decoration: Sets the decoration of the text like underline, overline, line-through, or none.
  • text-transform: Sets the transformation of the text like uppercase, lowercase, or capitalize.
  • letter-spacing: Sets the spacing between letters.
  • word-spacing: Sets the spacing between words.
  • font: Short hand that allows you to set the font-style, font-weight, font-size, and font-family in one line.
  • line-height: Sets the space between lines

Box Model Properties

  • border: Short hand that allows you to set the border-width, border-style, and border-color in one line. Applying this styling to a class and giving it to all other examples
  • border-[left,right,top,bottom]: Short hand to set the width, style, and color of the specified side of the border
  • border-radius: Sets the radius of the corners of the border of the element
  • border-width: Sets the width of the border of the element
  • border-style: Sets the style of the border of the element
  • padding:-(left, right, top, bottom) Sets the padding inside the border of the element
  • margin-(left, right, top, bottom): Sets the margin outside the border of the element

Sizing Properties

  • width: Sets the width of the element.
  • height: Sets the height of the element
  • max-width: Sets the maximum width of the element
  • max-height: Sets the maximum height of the element
  • min-width: Sets the minimum width of the element
  • min-height: Sets the minimum height of the element

List Properties

  • list-style-type: Sets the type of list item marker like disc, circle, square, or none.
  • list-style-image: Sets the image of the list item marker.
  • list-style-position: Sets the position of the list item marker like inside or outside.

Link Properties

  • You can style links using the a selector or class/id
  • You can also style their states like hover, visited, active
  • These can't be styled in-line and need to be styled using a psuedo-class (:hover, :visited, :active) selector in either the header or styling page

Forms

Alignment and Floating

text-align

Sets the alignment of the text like left, right, center, or justify.

float

Sets the position of the element like left, right, or none.

clear

Clears the float of the element like left, right, or both.

Box-sizing

box-sizing:border-box

When box-sizing is set to border-box, the width and height include the padding and border, making the element take up the full width and height of the container.

box-sizing:content-box

When box-sizing is set to content-box, the width and height only include the content, not the padding and border, making the element take up the width and height of the content.

Positioning

position:[VALUE] Adjusts element's position based on the input value

  • static: The default position of the element. It is positioned according to the normal flow of the document.
  • relative: The element is positioned according to the normal flow of the document, but can be offset with the top, right, bottom, and left properties.
  • absolute: The element is positioned according to the normal flow of the document, but can be offset with the top, right, bottom, and left properties.
  • fixed: The element is fixed to the viewport and does not move when the page is scrolled.
  • initial: The element is set to its initial value.
  • inherit: The element is set to the value of its parent element.

Background Image

background-image: url('background-image.jpg');

The background-image property is used to set the background image of an element.

background-position: [value];

The background-position property is used to set the position of the background image.

background-size: [value];

The background-size property is used to set the size of the background image. Repeats by default. Other values include cover and contain. Cover will scale the image to cover the entire element, while contain will scale the image to fit the entire element.

background-repeat: [value];

The background-repeat property is used to set whether the background image repeats. In this case, set it to 300px like above, but set it to no-repeat to prevent it from repeating.
