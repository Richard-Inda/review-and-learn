Positioning

position: static

  • The default position of the element. It is positioned according to the normal flow of the document.

position: relative

  • The element becomes a positioned ancestor for any absolutely positioned children.
  • The element is positioned relative to its normal position.
  • The element can be offset with the top, right, bottom, and left properties.

position: absolute

  • The element is positioned with an absolute positional value (exact values, top, right, bottom, left). The position is relative to the nearest positioned ancestor.
  • If no positioned ancestor is found, the position is relative to the viewport.

position: fixed

  • The element is fixed to the viewport and does not move when the page is scrolled.

position: initial

  • The element is set to its initial value.

position: inherit

  • The element is set to the value of its parent element.
