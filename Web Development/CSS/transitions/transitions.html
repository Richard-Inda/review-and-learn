Transitions

Transitions are used to animate the properties of an element. Animations are used to animate the properties of an element.

Properties of Transitions

  • transition-property: The property of the element to transition.
  • transition-duration: The duration of the transition.
  • transition-timing-function: The timing function of the transition.
  • transition-delay: The delay of the transition.

Transitionable Properties

  • background-color: The background color of the element.
  • background-position: The background position of the element.
  • border-color: The border color of the element.
  • border-width: The border width of the element.
  • border-spacing: The border spacing of the element.
  • bottom: The bottom of the element.
  • width: The width of the element.
  • height: The height of the element.
  • margin: The margin of the element.
  • padding: The padding of the element.
  • font-size: The font size of the element.
  • font-weight: The font weight of the element.
  • font-style: The font style of the element.
  • font-family: The font family of the element.
