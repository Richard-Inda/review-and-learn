Tables

Tag Full Name Purpose
thead Table Header Defines the header of the table. Contains the column names. Important for accessibility.
tbody Table Body Defines the body of the table. Contains the actual data. Important for accessibility.
tfoot Table Footer Defines the footer of the table. Used for summary of data. Important for accessibility.
tr Table Row Defines a row of the table. Contains one or more cells. Used to group data into rows.
td Table Data Defines a cell of the table. Contains the actual data. Used to group data into cells.
th Table Header Defines a header cell of the table. Contains the column names. Used to group data into header cells.
Above are tags Above are names Above are purposes
